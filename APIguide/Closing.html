Closing the Request

You've done it! Now I bet you want to close everything down, huh?

You've done the hard work already. Pat yourself on the back, and prepare for the easiest thing you'll learn today.


What do I do?

Every session you start needs to end. So, if you've used the "init" function (which we have), you better have a closing function. The closing function we will be using is the "curl_close" function.

Typically at the end of your PHP section, but definitely after your three previous request functions:

  1. curl_init
  2. curl_opt
  3. curl_exec

we will put in the "curl_close" function.

Now, you're done.


That was easy. What did it do?

The "curl_close($variable)" function does a few helpful things:

  1. Obviously, it closes the cURL session you started.
  2. It frees all resources.
  3. That "$curl" handle you made? It's been deleted.
